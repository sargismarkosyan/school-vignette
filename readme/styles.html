MSKH School Vignette

Installation and start

Add new design

Add .css or .less file on /public/stylesheets

Css selectors you need

    All class page

  • .classroomInfo a div whit classroom image, classroom name and description
  • .classroomInfo img classroom image
  • .student a div whit student image, the quote of student and a url to student's personal page
  • .student img student image
  • .studentShortInfo student name, the quote and url
  • .student h3 student name
  • .student p student quote
  • .student a student url

    • Single student page

  • .studentLongInfo all info of student
  • .studentInfo student image, name and quote
  • .studentInfo img student image
  • .studentNameAndQuote student name and quote
  • .studentNameAndQuote h3 student name
  • .studentNameAndQuote p student quote

You can use modern.less of simple.less as example

Start and test your design

Open config.json

{

    "username": "admin",

    "password": "admin",

    "styles": ["simple.css", "modern.css"]

}

Add your design name in the styles row.

{

    "username": "admin",

    "password": "admin",

    "styles": ["simple.css", "modern.css", "your-design-name.css"]

}

Save this file, and push to GitHub.

Change a design

Open .css or .less file in /public/stylesheets

Do your changes, save this file, and push to GitHub.

Remove a design

Remove .css and .less file from /public/stylesheets

Open config.json

{

    "username": "admin",

    "password": "admin",

    "styles": ["simple.css", "modern.css", "your-design-name.css"]

}

Remove design name from the styles row.

{

    "username": "admin",

    "password": "admin",

    "styles": ["simple.css", "modern.css"]

}

Save this file, and push to GitHub.
